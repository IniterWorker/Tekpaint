# Tekpaint

Team work project

## Lib

* SFML
* LibLapin overlay lib (Epitech)

## features

* Layer
* Square and Circle pencil
* Workflow (board hidde&show)
* Ellipse
* Drawline (breizhman)
* DrawCircle
* DrawSquare
* Draw (Square/Circle) with border
* Export format BMP
* Export format FOX (proprietary format)
* Load BMP format
* Load FOX format (proprietary format)
* Pipette
* Palette
* Alphachannel (bitwise)
